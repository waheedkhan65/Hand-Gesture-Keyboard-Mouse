# Hand Gesture Keyboard & Mouse

![Status](https://img.shields.io/badge/Status-Completed-brightgreen)

## Overview
The **Hand Gesture Keyboard & Mouse** project is an innovative solution that allows users to control their computer using hand gestures. Leveraging computer vision and machine learning, it detects hand gestures in real-time and maps them to keyboard and mouse actions. This touch-free interface enhances accessibility and provides a futuristic user experience.

---

## Features
- **Gesture Recognition:** Detects multiple hand gestures with high accuracy.  
- **Keyboard Control:** Map gestures to keyboard inputs for typing and shortcuts.  
- **Mouse Control:** Control cursor movement, clicks, and scrolling using gestures.  
- **Real-Time Processing:** Smooth and responsive gesture detection.  
- **Cross-Platform Compatible:** Works on all major operating systems supporting Python.

---

## Technologies Used
- **Programming Language:** Python  
- **Libraries & Frameworks:** OpenCV, Mediapipe, PyAutoGUI, NumPy  
- **Machine Learning:** Hand gesture detection models for accurate tracking  
- **Interface:** Simple GUI for configuration and real-time feedback

## Installation
1. **Clone the repository:**
```bash
git clone https://github.com/waheedkhan65/Hand-Gesture-Keyboard-Mouse.git
cd Hand-Gesture-Keyboard-Mouse


2. **Create a virtual environment:**

```bash
python -m venv .venv
```

3. **Activate the virtual environment:**

* **Windows:**

```bash
.venv\Scripts\activate
```

* **Linux/Mac:**

```bash
source .venv/bin/activate
```

4. **Install dependencies:**

```bash
pip install -r requirements.txt
```

5. **Run the application:**

```bash
python main.py
```

---

## Usage

1. Launch the application.
2. Position your hand in front of the camera.
3. Perform gestures to control keyboard and mouse actions.
4. Adjust settings in the GUI for sensitivity or gesture configuration.
